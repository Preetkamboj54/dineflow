# Plan 09-03: Admin Dashboard & Restaurant Components Migration

## Objective

Migrate `AdminDashboard.jsx` and all 5 components in `client/src/components/` to use Tailwind utility classes. These files use a mix of Tailwind (already written inline) and custom CSS class references from the old `style.css`. The goal is to remove all remaining references to now-deleted utility classes and ensure only the preserved `@layer components` class names are used.

---

## Class Replacement Reference

Same mappings as Plan 09-02:

| Old class | Tailwind replacement |
|---|---|
| `text-muted` | `text-[var(--text-muted)]` |
| `text-primary` | `text-[var(--primary)]` |
| `text-secondary` | `text-[var(--secondary)]` |
| `text-danger` | `text-[var(--danger)]` |
| `container` (layout wrapper) | `max-w-[1200px] mx-auto my-8 px-6` |
| `shadow` | `shadow-md` |
| `rounded-lg` | `rounded-xl` |

Keep: `card`, `btn-primary`, `btn-secondary`, `btn-sm`, `btn-danger`, `stats-card`, `kanban-board`, `kanban-column`, `menu-item`, `form-group`
