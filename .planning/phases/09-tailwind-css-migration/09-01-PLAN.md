# Plan 09-01: CSS Variable Bridge, App.jsx Nav & Auth Pages

## Objective

Strip `style.css` of all utility/component classes that duplicate Tailwind, retaining only:
1. The `@import "tailwindcss"` directive
2. Google Fonts import
3. CSS custom properties (design tokens) in `:root`
4. A thin `@layer base` block for global element resets (body, headings)
5. Tailwind `@layer components` blocks for the ~6 shared semantic classes that remain genuinely reusable across many files (`btn-primary`, `btn-secondary`, `btn-sm`, `btn-danger`, `card`, `auth-container`)

Then rewrite `App.jsx` header/nav and both auth pages (`Login.jsx`, `Register.jsx`) to use pure Tailwind utility classes.
