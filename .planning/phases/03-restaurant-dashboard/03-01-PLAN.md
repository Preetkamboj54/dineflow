# Plan 03-01: Menu and reservation management endpoints and UI

## Goal
Implement the core CRUD endpoints for menu items, a simple list view for table reservations, and the frontend React components to manage them inline.

## Requirements
- REST-01: Restaurant staff can add, edit, or delete menu items and toggle availability
- REST-05: Restaurant staff can view upcoming table reservations and customer details

## Context
- **Menu Management:** Inline editing on the menu page.
- **Reservations:** Simple list format.
- **Auth:** Must use `authMiddleware.verifyToken` and `verifyRole('Restaurant')`.
